### 142. 环形链表 II
给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。请用空间复杂度为O(1)的方法解决。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。注意，pos 仅仅是用于标识环的情况，并不会作为参数传递到函数中。
### 样例
#### 输入
```
head = [3,2,0,-4], pos = 1
head = [1,2], pos = 0
head = [1], pos = -1
```
#### 输出
```
返回索引为 1 的链表节点
返回索引为 0 的链表节点
返回 null
```
